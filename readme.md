# Project README

## Overview

This project is a Python environment set up with various packages commonly used for development, debugging, and data analysis. It includes Jupyter support, visualization tools, and utility libraries.

## Requirements

All dependencies are listed in the `requirements.txt` file. To install them, run:

```bash
pip install -r requirements.txt
```

## Features

* Jupyter Notebook and JupyterLab support
* Debugging tools (e.g., `debugpy`, `asttokens`)
* Visualization support via `matplotlib-inline`
* Rich text rendering with `rich` and `Pygments`

## Usage

1. Clone the repository
2. Install dependencies
3. Launch Jupyter Notebook:

   ```bash
   jupyter notebook
   ```
